# Starter Code

This directory stores downloadable starter/template code for student exercises.

It is intentionally excluded from generated navigation and sitemap output. Link to files here explicitly from the relevant content page.

Recommended layout:

```text
Content/Code/<topic-slug>/
  java/
  cpp/
  python/
  downloads/
```

Use lowercase URL-friendly topic slugs, for example:

```text
Content/Code/stacks/
Content/Code/queues/
Content/Code/linked-lists/
Content/Code/binary-search-trees/
```

For one-file starters, link directly to the source file. For multi-file starters, keep the source files in the language directory and provide per-language zip files in `downloads/`.

Do not put solution files here unless the site policy changes.
